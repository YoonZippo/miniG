# 🎮 miniG - 프리미엄 미니게임 디스코드 봇

**miniG**는 친구들과 함께 디스코드에서 즐길 수 있는 고퀄리티 심리 게임 봇입니다. 
라이어 게임과 스파이폴을 기반으로 한 긴장감 넘치는 게임 플레이와, AI로 생성된 화려한 시각적 효과를 제공합니다.

---

## ✨ 주요 기능

### 1. 🏅 전적 및 프로필 시스템
단순한 게임 플레이를 넘어, 여러분의 기록을 영구적으로 저장합니다.
- `!프로필`: 나의 라이어 게임/스파이폴 승률 및 판수 확인.
- `!랭킹`: 서버 내 진정한 심리전의 고수가 누구인지 랭킹으로 확인.

### 2. 🕵️ 라이어 게임 (Liar Game)
- **자동 턴 관리**: 플레이어 답변 시 자동으로 다음 사람에게 턴이 넘어갑니다.
- **턴 타이머**: 한 사람당 20초의 제한시간으로 속도감 있는 게임 진행.
- **바보 모드**: 라이어도 가짜 제시어를 받는 특별한 모드 지원.

### 3. 🕵️‍♂️ 스파이폴 (Spyfall)
- **고화질 이미지 연동**: 시민 역할 배정 시, 해당 장소의 이미지가 DM으로 함께 전송됩니다.
- **긴박한 알림**: 토론 종료 직전(30/10/5초 전) 자동 카운트다운 알림.
- **음성 채널 자동 접속**: 게임 시작 시 봇이 음성 채널에 함께 들어와 분위기를 조성합니다.

---

## 🛠️ 주요 명령어

| 명령어 | 설명 |
| :--- | :--- |
| `!시작` | 봇의 메인 메뉴를 출력합니다. (게임 선택 및 업데이트 확인) |
| `!프로필` | 나의 게임 전적 정보를 출력합니다. |
| `!랭킹` | 서버 내 전체 전적 랭크를 확인합니다. |
| `!종료` | 진행 중인 게임과 음성 채널 연결을 강제 종료합니다. |
