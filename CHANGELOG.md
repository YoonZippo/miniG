# 📋 miniG 봇 업데이트 기록

### [v1.3.1] - 2026-02-25
- **🏗️ 프로젝트 구조 대규모 리팩토링**: 
  - `gameBot/` 및 `musicBot/` 디렉토리로 소스 코드를 분격화하여 관리 효율성을 높였습니다.
  - 각 봇이 독립된 Docker 컨테이너에서 구동되도록 설정을 최적화했습니다.
  - 최상위 디렉토리를 깔끔하게 정리하여 공용 파일(README, CHANGELOG 등)만 남겨두었습니다.

### [v1.3.0] - 2026-02-25
- **🏅 전적 및 랭킹 시스템 도입**: SQLite 기반 데이터베이스를 연동하여 `!프로필`과 `!랭킹` 기능을 추가했습니다.
- **🎨 UI 가독성 개선**: 메인 메뉴의 게임 버튼에 권장 인원수(3인~ 등) 안내를 추가하여 편의성을 높였습니다.
- **⏱️ 게임 타이머 고도화**: 
  - 라이어 게임: 플레이어당 20초 제한시간 추가 (미입력 시 자동 턴 스킵).
  - 스파이폴: 토론 종료 30초/10초/5초 전 자동 알림 시스템 추가.
- **🎨 스파이폴 시각적 강화**: 게임 시작 시 시민들에게 해당 장소의 **고화질 AI 생성 이미지(10종)**가 DM으로 함께 전송됩니다.
- **🤖 메인 엔진 통합**: 새로운 기능들을 `miniG.py`와 `StatsCog`로 통합하여 안정성을 높였습니다.

### [v1.2.0] - 2026-02-25
- **라이어 게임 카테고리 확장**: '물건', '취미/스포츠' 카테고리가 새롭게 추가되었습니다.
- **UI 개선**: !시작 메뉴에 '업데이트 목록' 버튼이 추가되었습니다.
- **편의 기능**: 라이어 게임 로비에서 현재 지원하는 카테고리를 미리 볼 수 있습니다.
- **안정성 개선**: 깃(Git) 병합 충돌 해결 및 코드 구조 최적화.

### [v1.1.0] - 2026-02-24
- **스파이폴(Spyfall) 게임 추가**: 새로운 심리 게임 '스파이폴'이 추가되었습니다.
- **바보 라이어 모드**: 라이어에게도 가짜 제시어를 주는 새로운 모드가 추가되었습니다.
- **서버 배포**: 구글 클라우드(GCE) 환경 배포 인원 및 Docker 설정 완료.

### [v1.0.0] - 2026-02-23
- **프로젝트 시작**: 기본적인 라이어 게임 기능 구현.
- **자동 턴 시스템**: 채팅 기반의 자동 턴 전환 및 메시지 임베드 처리.
